How to comment 
Your feedback is welcome on either version.
### To add a comment:
1. Read the PDF you want to comment on
2. Go to the [Issues tab](https://github.com/GlobalStewardship2026/global-stewardship-charter-2026-public-review/issues)
3. Click **"New Issue"**
4. Use label: `charter-feedback`
5. Mention which version (April 15 or April 18) and which section/page
6. Submit your comment
